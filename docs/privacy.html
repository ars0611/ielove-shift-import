プライバシーポリシー

Shift Import（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、 個人情報および Google ユーザーデータを適切に取り扱います。

1. 取得する情報

  • ユーザーが許可した Google アカウントの基本的な認可情報
  • ユーザーが指定した Google スプレッドシート内のシフト情報
  • Google カレンダーへ予定を登録するために必要なカレンダー情報
  • 本アプリ内の設定情報

2. 利用目的

  • Google スプレッドシートからシフト情報を読み取るため
  • Google カレンダーへ予定を登録、更新するため
  • ユーザー設定に応じた動作を提供するため
  • 不具合調査およびサービス改善のため

3. データの保存について

本アプリは、必要最小限の情報のみを利用します。 保存する情報がある場合は、ブラウザ内または拡張機能の保存領域に限定します。 サーバーへ恒久的に保存する場合は、その内容を別途明示します。

4. 第三者提供について

本アプリは、法令に基づく場合を除き、取得した情報を第三者へ提供しません。

5. Google ユーザーデータの取り扱い

本アプリは、Google Sheets API および Google Calendar API を通じて取得したデータを、 シフト情報の読み取りおよびカレンダー登録の目的にのみ使用します。 これらのデータを広告目的で利用することはありません。

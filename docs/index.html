Chrome 拡張機能

Shift Import

Shift Import は、Google スプレッドシートに記載されたシフト情報を読み取り、 Google カレンダーへ予定として登録するための Chrome 拡張機能です。

ユーザーが開いているシフト表を起点に、必要な情報を取得し、カレンダー登録を補助します。

主な機能

  • Google スプレッドシート上のシフト情報の読み取り
  • 対象ユーザーのシフト抽出
  • Google カレンダーへの予定登録
  • 設定情報に応じた登録内容の調整

利用する Google サービス

  • Google Sheets API
  • Google Calendar API

データの取り扱い

Shift Import は、シフト情報の読み取りと Google カレンダーへの予定登録のために、 ユーザーが許可した Google アカウントのデータへアクセスします。

詳しくは プライバシーポリシー をご確認ください。
